## Hearthstone Deck Builder ##

# ---------------------------------------- Idea -------------------------------------------------- #

The main idea of the application is a deck builder for the popular card collecting game Hearthstone. 
Without the registratration the following features are available to the user: creating and modifying
a new deck, importing a deck using a deck code and copying a deck code. The logged-in users also 
benfit from an opportunity to store decks for later use, annotate them with comments and set rating 
as well as access the previously created decks.
